# TripleTen Art Gallery web_project_homeland

Proyecto de desarrollo web responsivo basado en un diseño Figma.
Este sitio muestra cómo adaptar un diseño a diferentes resoluciones, garantizando una experiencia óptima en escritorio, tablet y móviles pequeños.

# Descripción

TripleTen Art Gallery presenta una galería de historias y arte de personas de distintas partes del mundo. El sitio se centra en la presentación de contenido visual y textual siguiendo principios de diseño responsivo y buenas prácticas de CSS.

Bloques principales:

- **Header**: logo centrado y navegación (si se agregara en el futuro)
- **Lead**: imagen principal con título y subtítulo
- **Intro**: texto explicativo y cita de Joseph Beuys
- **Photo-grid**: una pequeña galeria de fotos
- **Places**: información desglosada sobre el lugar de origen de los diferentes artistas del sitio web.
- **Footer**: información de copyright

---

## Tecnologías utilizadas

- HTML5
- CSS (con metodología BEM)
- Normalize.css
- Diseño responsivo
- Fuentes personalizadas
- Grid
- Box-shadow

## Funcionalidades destacadas

- Uso de `flexbox` para centrar elementos y organizar bloques
- `box-sizing: border-box` aplicado globalmente para evitar scroll horizontal
- Ajustes tipográficos y de padding con media queries
- Imágenes y logos adaptativos con `max-width: 100%` para mantener proporción
- Incorporación de grid para darle un desgloce más atractivo y práctico al diseño.
- Diseño responsivo, e interactivo para pantallas de 1280 px.
- Incorporación de efecto visual aplicando box-shadow.
- Footer siempre visible y centrado en todos los tamaños de pantalla
